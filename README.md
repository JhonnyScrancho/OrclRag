## Architecture

The system consists of several components:

1. **Frontend Layer**
   - Streamlit-based user interface
   - Real-time interaction
   - Progress tracking

2. **Agent Layer**
   - Multiple analyzer agents
   - Synthesis agent
   - Load balancing

3. **Processing Layer**
   - Document chunking
   - Embedding generation
   - Vector storage

4. **Database Layer**
   - Vector database
   - Metadata storage
   - Query optimization
